# 实验四：函数近似：插值与拟合

## 物理背景

在真实的物理实验和数值模拟中，我们通常只能获得离散的数据点。为了从这些离散数据中恢复出连续的物理规律，我们需要用到**函数近似**技术：

1. **插值 (Interpolation)**：要求曲线**严格穿过**每一个已知的数据点。它适用于数据精度极高（如天文光谱的重采样对齐），且我们主要关心中间点数值的情况。
2. **拟合 (Fitting)**：不要求曲线穿过所有点，而是寻找一个参数化模型来**最好地反映整体趋势**。这在处理带有实验误差的数据（如中子共振截面）时尤为关键。

本周我们将通过三个并行任务和一个进阶挑战，深入理解插值与拟合在不同物理场景下的应用。

**1. 任务 A：漏洞破译 (Runge 现象物理陷阱)**
- **位置**: `lab1_core/src/runge_bad_ai.py`
- **背景**: 某个 AI 试图用 10 阶拉格朗日多项式对 11 个均匀分布的数据点进行全局插值 $f(x) = \frac{1}{1 + x^2}$。
- **目标**: 运行代码，观察区间边缘灾难性的振荡现象。用 Prompt 引导 AI 或手动将插值方法替换为**分段三次样条插值 (`CubicSpline` 或 `interp1d(kind='cubic')`)**，使误差降低到合理的物理范围。

**2. 任务 B：算法重构 (共振截面非线性拟合)**
- **位置**: `lab1_core/src/resonance_fit.py`
- **背景**: 实验室测得了一组中子共振截面数据，需要从中提取共振峰位置 $E_r$ 和半高全宽 $\Gamma$。截面服从 Breit-Wigner 分布：
  $$
  f(E)=\frac{f_r}{(E-E_r)^2+\Gamma^2/4}
  $$
- **目标**: 使用 `scipy.optimize.curve_fit` 实现非线性拟合，并返回物理参数。
**提示：**
- 可根据截面峰值附近的数据估计 $E_r$ 初值；
- 峰宽可由半高宽粗略估计；
- 建议为参数提供合理初值 `p0`，必要时使用 `bounds` 限制参数范围。

**3. 任务 C：物理洞察 (加权最小二乘拟合)**
- **位置**: `lab1_core/src/weighted_fit.py`
- **背景**: 在 Task B 中，我们假定所有实验数据点的可信度是一样的。但在真实物理实验中，每个数据点都有其测量误差 $\sigma_i$。误差越大的点，其在拟合中的权重应该越小。
- **目标**: 在 `curve_fit` 中引入 `sigma` 参数，实现**加权最小二乘拟合**。对比加权前后的 $E_r$ 和 $\Gamma$ 参数变化，在报告中探讨这反映了什么物理直觉。

### 🚀 Lab 2: 复杂系统探索 (进阶) - 满分 30 分
**位置**: `lab2_bonus/src/galaxy_spectra.py`
- **背景**: 在天文学中，两台不同的望远镜对同一个星系进行了观测，但它们的**波长采样网格不同**。为了合并这两组数据，我们必须将它们重采样到同一个标准的波长网格上。
- **目标**: 利用 `scipy.interpolate.interp1d` (建议使用三次样条 `cubic`) 在 A 与 B 的重叠波长范围内构造统一标准网格，再将 A 与 B 同时重采样到该网格后进行合成。这是插值在科研中非常高阶且真实的运用场景。
